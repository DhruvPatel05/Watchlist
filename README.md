# 🎬 Watchlist

Watchlist is a modern SwiftUI application that helps users keep track of movies they want to watch in the future.

The app allows users to create a personal movie watchlist, categorize movies by genre, and even randomly select a movie whenever they can't decide what to watch next.

---

## ✨ Features

- ➕ Add movies to your watchlist
- 🎭 Categorize movies by genre
- 🎲 Random movie picker
- 🗑️ Delete movies with swipe actions
- 💾 Persistent local storage using SwiftData
- 🎬 Beautiful onboarding experience
- 📱 Clean and modern SwiftUI interface
- ⚡ Fast and responsive performance

## 🚀 Installation

Clone the repository:

```bash
git clone git@github.com:DhruvPatel05/Watchlist.git
```

Open the project:

```bash
open Watchlist.xcodeproj
```

Build and run using an iPhone Simulator or physical device.

---

## 📱 User Flow

1. Launch the app
2. Add movies to your watchlist
3. Select a movie genre
4. Save the movie
5. View all saved movies
6. Use the randomizer button to pick a movie
7. Delete movies when no longer needed
